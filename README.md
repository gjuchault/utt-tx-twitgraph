# twit-graph

A educational tool to map a twitter user relationships

## Required software

  - node (version 8 minimum)
  - neo4j

## Installation

```sh
git clone https://github.com/BabigaBirregah/TwitterSocialGraphCollector.git twit-graph
cd twit-graph

# edit config.json

yarn build
```

## Running

```sh
yarn start

# or

yarn dev
```
